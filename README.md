# 🛡️ UPI Fraud Detector (Hybrid AI)

An AI-powered web application that detects fraudulent financial messages using a hybrid approach combining rule-based logic and machine learning.

## 🚀 Features

* Detects scam messages (OTP, phishing, UPI fraud)
* Identifies legitimate bank transaction messages
* Hybrid system:

  * Rule-based detection
  * ML model (TF-IDF + Logistic Regression)
* Confidence score for predictions
* Real-time analysis via web app

## 🧠 Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas

## 🔥 How It Works

1. Rule-based system detects high-risk fraud patterns
2. Bank-safe detection prevents false positives
3. ML model classifies unknown patterns
